## ADDED Requirements

### Requirement: README uses English-only text
All section headers, body text, and inline callouts in `README.md` SHALL be written in English. No mixed-language content is permitted.

#### Scenario: Developer reads section headers
- **WHEN** a developer opens `README.md`
- **THEN** all section headers (e.g. "Getting Started", "Prerequisites", "Installation", "Documentation", "Built with") are in English with no Spanish text present
